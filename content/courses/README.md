# 教材定義（`courses/`）

カテゴリ別にフォルダを分け、その下に **教材ごと** のサブフォルダと `course.json` を置く。

| フォルダ | 用途 |
|----------|------|
| `英文法/` | 1文 CSV の `id`（例 `G1-01-001`）とステージ（`G1-01` など）で束ねる教材 |
| `リスニング/` | リスニング用。**英文法と同じ 6 教材**（`mirror_of` で対応）。詳細は `リスニング/README.md` |
| `英単語/` | 語彙教材用（`text/vocabulary/…` と対応させる想定） |
| `イディオム/` | イディオム用（今後追加） |

全体の目次はひとつ上の **`catalog.json`** の `courses` に列挙する。

## 新しい教材を追加するとき

1. 指示文（ステージ名と使う文の id / 接頭辞）を `指示テンプレート.md` の形式で渡す。
2. `course.json` を追加し、`catalog.json` に `course_id` と `relative_path` を追記する。
