# 教材データの依頼フォーマット（コピー用）

以下をそのまま埋めて送ってください。ステージは **1行に1つ**、`ステージ名` と **`文IDの接頭辞`**（CSV の `id` が `G1-01-001` のとき接頭辞は `G1-01`）をカンマ区切りにします。

---

**カテゴリ**: （例: 英文法 / リスニング / 英単語 / イディオム）

**教材名**: （例: やり直し英文法 基礎①）

**教材ID（英数字スラッグ）**: （例: `yarinaoshi_basic1`）。未指定なら教材名から自動で付けます。

**本文**（相対パス）: `１文データ/one_sentence/text/one_sentence.csv` など

**音声 en**（相対パス）: `１文データ/one_sentence/audio/en/`

**音声 ja**（任意）: `１文データ/one_sentence/audio/ja/`

**ステージ一覧**（ステージ名,使用する文の id 接頭辞）:

```
自分・相手・ものを表現する,G1-01
動作を表現する,G1-02
尋ねる・質問する,G1-04
人・ものを指す,G1-07
```

---

接頭辞に一致する **全文**（例: `G1-01-001` … `G1-01-210`）がそのステージに含まれます。個別の ID だけにしたい場合は別途「列挙で指定」と書いてください。
